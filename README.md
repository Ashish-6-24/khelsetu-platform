🏆 KhelSetu Platform

A modern real-time sports tournament management platform built with React, TypeScript, and Vite.

✨ Features: 
Real-time multi-sport scoring
Tournament & fixture management
Live standings and analytics
Player & team management
OBS-ready broadcast overlays
Offline-first synchronization
Role-based access control
Fully responsive UI/UX

🛠 Tech Stack:
React 19 + TypeScript
Vite 8
TailwindCSS 4
Zustand
TanStack Query
React Router v7
Framer Motion
Socket.io
Vitest + Playwright

🚀 Getting Started:
Install Dependencies
npm install
Start Development Server
npm run dev
